FieldPrime

FieldPrime is a software suite designed to ease the process of manual collection and collation of data. For example the collection of trait score data in agricultural trials.
The software consists of the FieldPrime Android app - used for the data collection. And the FieldPrime Server - used to store and manage the trial data.
