News

  • 03/6 2015

    Alternates to "Row" and "Column"Michael Kirk

    You can now use alternate names for "Row" and "Column" in your trials: Under the trial "Properties" tab enter alternate values appropriate for your needs (for example "Range" and "Run"). These options are also there on the New Trial page, so when create a trial via a CSV file, the file can contain your preferred names.

    These options have to be set on a per trial basis - if not set, the default values "Row" and "Column" still apply. In the future these will probably become configurable on a project basis - to apply to all trials in a project.

    Note that to see the new names on the app itself, you will need to update to the latest app version - 1.0.4.

  • 03/6 2015

  • 25/5 2015

    New app version v1.0.2Michael Kirk

    An updated app version is available. This is not significantly different from the previous version (v1.0.1) except for a bug fix. In the previous version dynamic traits were not functioning properly on the app. You couldn't navigate between the multiple scores in a dynamic trait if that trait was the first one in the set of scoring traits. This is fixed in the new version.

  • 18/3 2015

    ScoreSet plotsMichael Kirk

    On the page for any numeric score set sample (accessed from the ScoreSets tab) there are now a few plots shown for the data: A box and whiskers plot, a histogram, and a scatter plot for which you can specify any numerical attribute in the trial to plot against. Feedback on these, or ideas for other plots are welcome..

  • 12/1 2015

    ***REMOVED*** usersMichael Kirk

    The FieldPrime server now supports ***REMOVED*** users. What this means is that instead of everyone in a project using the same project name and password to login, you can instead use (if you have one) your CSIRO ***REMOVED*** identity. To allow a ***REMOVED*** user to login using their own ident and password, first you must login with the usual project login, go to administration page and add users in the "Manage ***REMOVED*** Users" section. This also means that if you have access to multiple projects, you can access all of them via a single ***REMOVED*** login (via a dropdown list of projects you have access to at the top of the FieldPrime pages).

  • 18/6 2014

    Deleting TrialsMichael Kirk

    User can now delete unwanted trials. When you select a trial, there is a "Delete This Trial" button near the top of the page. You will be asked to confirm, and re-enter your password, just to be sure. Note deleting a trial means all scores, traits, etc created for that trial will be lost - so use with caution!

  • 7/2 2014

    FieldPrime app version 0.1 released Michael Kirk

    New version of the app with quite a few changes, including:

    • When you open a trial, there is now a browse mode as well as a score mode, the browse mode allows you to view all the current scores and notes on the device (or some subset of them you can select), and the score mode is as it was before. You swipe the screen left or right to switch between these.
    • The number of property widgets can be modified. By default it is still 2, but you can change this from a menu accessed by touching the FieldPrime logo in the top left of the screen.
