News

  • 12/1 2015

    Makeover (and ***REMOVED*** users)Michael Kirk

    FieldPrime has had a makeover! with some inspired graphic design provided by Raj Gaire. In addition, the new version now supports ***REMOVED*** users. What this means is that instead of everyone in a project using the same project name and password to login, you can instead use (if you have one) your CSIRO ***REMOVED*** identity. To allow a ***REMOVED*** user to login using their own ident and password, first you must login with the usual project login, go to administration page and add users in the "Manage ***REMOVED*** Users" section. This also means that if you have access to multiple projects, you can access all of them via a single ***REMOVED*** login (via a dropdown list of projects you have access to at the top of the FieldPrime pages).

  • 9/9 2014

    Version 0.2.3Michael Kirk

    New app version (0.2.3) is available from the Download App page. There are a few bug fixes and improvements here. One improvement is that now to export, update, or delete a trial, you can do a long press on the trial name in the trial list on the main FieldPrime screen. (This is a bit more convenient than using the menus).

    As usual you may need to delete the FieldPrime folder on your device. So make sure you have uploaded everything before upgrading!

  • 1/8 2014

    Version 0.2.1Michael Kirk

    New app version (0.2.1) available from the Download App page. There are a few small changes in here, but in particular the upload of photos from the app has been improved. This version does include changes to the database on the app. Which means before installing, please upload any data you haven't yet uploaded, and then delete the FieldPrime database on the device.

  • 18/6 2014

    Deleting TrialsMichael Kirk

    User can now delete unwanted trials. When you select a trial, there is a "Delete This Trial" button near the top of the page. You will be asked to confirm, and re-enter your password, just to be sure. Note deleting a trial means all scores, traits, etc created for that trial will be lost - so use with caution!

  • 25/2 2014

    Please add your contact details Michael Kirk

    For users with a login, there's now a place to add a contact name and email address for the project (under the "Profile/Passwords" link on the left hand side once you log in). Please add a name and email address here so that we can contact you if necessary.

  • 14/2 2014

    New website look Michael Kirk

    Added this news section, contact page and so on.

  • 7/2 2014

    FieldPrime app version 0.1 released Michael Kirk

    New version of the app with quite a few changes, including:

    • When you open a trial, there is now a browse mode as well as a score mode, the browse mode allows you to view all the current scores and notes on the device (or some subset of them you can select), and the score mode is as it was before. You swipe the screen left or right to switch between these.
    • The number of property widgets can be modified. By default it is still 2, but you can change this from a menu accessed by touching the FieldPrime logo in the top left of the screen.
